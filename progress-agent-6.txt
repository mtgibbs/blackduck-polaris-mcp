## Codebase Patterns

### Reports API Module Pattern
- Reports API uses standard JSON (Content-Type: application/json), NOT vendor media types
- Base path: /api/insights/ for all report-related endpoints
- Use `getAllOffset` for paginated list endpoints
- Service layer wraps API functions with Options interfaces
- API functions use client.get(), client.fetch(), etc. from PolarisClient
